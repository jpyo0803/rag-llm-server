# 포켓몬 속성 상성 규칙

- 불(파이리)은 풀(이상해씨)에게 강하지만, 물(꼬부기)에게 약하다.
- 물(꼬부기)은 불(파이리)에게 강하지만, 풀(이상해씨)에게 약하다.
- 풀(이상해씨)은 물(꼬부기)에게 강하지만, 불(파이리)에게 약하다.
- 전기(피카츄)는 물(꼬부기)에게 강하지만, 풀(이상해씨)에게 약하다.
- 전기는 불(파이리)과 상성 중립이다.
- 동일한 속성 간 전투는 서로 중립이다.

요약:
불 → 강: 풀 / 약: 물  
물 → 강: 불 / 약: 풀  
풀 → 강: 물 / 약: 불  
전기 → 강: 물 / 약: 풀
